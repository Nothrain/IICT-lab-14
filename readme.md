## Introduction
This is a site created for a university (IOBM) assignment.

## Technologies Used

- **HTML5** - Semantic structure with proper tags (header, nav, main, section, footer)
- **CSS3** - External stylesheet with responsive design and modern styling
- **JavaScript (ES6)** - DOM manipulation, form validation, and interactive features
- **GitHub Pages** - Deployment and hosting

## Features

### HTML Structure
- Semantic HTML5 doctype and document structure
- Navigation menu with smooth scrolling
- Four main content sections (Home, About, Menu, Contact)
- Contact form with multiple input types
- Proper use of semantic tags throughout

### CSS Implementation
- **External CSS:** Primary styling in `styles.css`
- **Internal CSS:** Page-specific styles in `<style>` tags within the About section
- **Inline CSS:** Hero section subheadline demonstration
- Responsive design for mobile, tablet, and desktop
- CSS Grid and Flexbox layouts
- Hover effects and transitions
- Custom color scheme with brand consistency

### JavaScript Functionality
- **Form Validation:** Real-time validation for reservation form
  - Name validation (minimum 2 characters)
  - Email format validation
  - Phone number validation
  - Date validation (no past dates)
  - Guest number validation (1-12)
- **Interactive Features:**
  - Smooth scrolling navigation
  - Menu item hover effects (DOM manipulation)
  - Active navigation highlighting on scroll
  - Dynamic success/error messages
  - Automatic past date prevention
- **Event Handling:** Click, submit, scroll, and mouse events
